# Plant Cabinet
### Organize and manage your plants during quarantine.
Sign up for an account, create plants, add reviews, and monitor your plant collection.
## Installation
    1. Clone this repo
    2. $ bundle install
    3. $ rails s
## Usage
    Go to localhost:3000/ in your browser
